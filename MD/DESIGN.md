# Design-System

Dieses Dokument definiert die verbindlichen Designrichtlinien für das gesamte Projekt.

Alle neuen Seiten, Komponenten und Änderungen müssen sich an diesen Vorgaben orientieren.

---

# Designphilosophie

Die Webseite soll modern, hochwertig, elegant und minimalistisch wirken.

Der Fokus liegt auf Vertrauen, Klarheit und einer hohen Conversion-Rate.

Das Design soll zeitlos sein und nicht wie ein typisches KI-Template oder eine Baukasten-Webseite aussehen.

Weniger ist mehr.

Jedes Element benötigt einen klaren Zweck.

---

# Designprinzipien

* Mobile First
* Klare visuelle Hierarchie
* Große Weißräume
* Ruhige Gestaltung
* Hohe Lesbarkeit
* Schnelle Ladezeiten
* Barrierearme Bedienung
* Wiederverwendbare Komponenten
* Einheitliches Erscheinungsbild

# Formulare

Alle Formulare besitzen

* Labels
* ausreichend große Eingabefelder
* klare Fehlermeldungen
* große Buttons

Desktop

maximal zwei Spalten

Mobil

immer eine Spalte

Der primäre Call-to-Action erfolgt über ein Formular.

---

# Karten

Karten wirken hochwertig und ruhig.

Eigenschaften

* Weißer Hintergrund
* Dezenter Schatten
* 16px Rundung
* Innenabstand 32px
* Einheitliche Höhe wenn möglich

Keine überladenen Karten.

---

# Navigation

Navigation bleibt schlicht.

* Logo links
* Menü rechts
* maximal sechs Menüpunkte
* deutlicher CTA-Button
* Sticky Navigation

Mobil

Hamburger-Menü

---

# Hero

Jede Landingpage beginnt mit einem Hero-Bereich.

Hero enthält:

* starke Überschrift
* kurze Unterüberschrift
* primären CTA
* optional Vertrauenselemente
* hochwertiges Bild oder Illustration

Kein überladener Hero.

---

# Icons

Einheitlicher Stil.

Keine gemischten Icon-Sets.

Icons dienen ausschließlich der Orientierung.

---

# Bilder

Nur hochwertige Bilder verwenden.

Keine Pixelgrafiken.

Keine Cliparts.

Keine generischen Stockfotos mit künstlichem Eindruck.

---

# Animationen

Animationen unterstützen den Nutzer.

Erlaubt

* Fade
* Slide
* Hover
* leichte Skalierung

Dauer

150–300 ms

Keine übertriebenen Animationen.

# Barrierefreiheit

Alle Bilder besitzen Alt-Texte.

Ausreichende Farbkontraste.

Buttons sind per Tastatur erreichbar.

Focus-Zustände sind sichtbar.

Animationen respektieren "prefers-reduced-motion".

---

# Performance

* Lazy Loading für Bilder
* Optimierte Bildgrößen
* Keine unnötigen Bibliotheken
* Wiederverwendbare Komponenten
* Möglichst wenig JavaScript

---

# Verbotene Designmuster

Nicht verwenden:

* Neonfarben
* bunte Farbverläufe
* Glassmorphism als Hauptstil
* riesige Schatten
* riesige Rundungen
* Emojis als Designelemente
* Cliparts
* zufällige Formen
* überladene Hintergründe
* automatisch generierte Standardtexte
* Lorem Ipsum
* erfundene Bewertungen
* erfundene Zahlen
* mehr als drei Button-Designs
* unterschiedliche Kartendesigns auf einer Seite
* unterschiedliche Schriftarten innerhalb derselben Seite

---

# Qualitätsstandard

Vor jeder Fertigstellung prüfen:

* Mobile Darstellung
* Tablet-Darstellung
* Desktop-Darstellung
* Einheitliche Abstände
* Einheitliche Farben
* Einheitliche Typografie
* Konsistente Buttons
* Lesbarkeit
* Conversion-Fokus
* Performance
* Barrierefreiheit
* Keine Layoutfehler
* Keine überflüssigen Elemente

Im Zweifel gilt:

**Einfacher, ruhiger und hochwertiger schlägt komplex, laut und verspielt.**
